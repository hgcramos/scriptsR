E agora vamos modificar a fun√ß√£o, para apresentar diferentes elementos em dierentes partes do painel.

Primeiro, vamos criar a fun√ß√£o panel.hist (copiando e colando do help da fun√ß√£o pairs):

panel.hist <- function(x, ...)
{
    usr <- par("usr"); on.exit(par(usr))
    par(usr = c(usr[1:2], 0, 1.5) )
    h <- hist(x, plot = FALSE)
    breaks <- h$breaks; nB <- length(breaks)
    y <- h$counts; y <- y/max(y)
    rect(breaks[-nB], 0, breaks[-1], y, col = "cyan", ...)
}

Agora, vamos fazer o mesmo com a fun√ß√£o panel.cor:

panel.cor <- function(x, y, digits = 2, prefix = "", cex.cor, ...)
{
    usr <- par("usr"); on.exit(par(usr))
    par(usr = c(0, 1, 0, 1))
    r <- abs(cor(x, y))
    txt <- format(c(r, 0.123456789), digits = digits)[1]
    txt <- paste0(prefix, txt)
    if(missing(cex.cor)) cex.cor <- 0.8/strwidth(txt)
    text(0.5, 0.5, txt, cex = 2)
}

Na minha vers√£o eu modifiquei a forma do argumento cex, pois n√£o gosto dos n√∫meros com tamanho proporcional ao valor da correla√ß√£o.

Agora vamos fazer o novo painel!

pairs(fisqui, diag.panel=panel.hist, upper.panel = panel.cor)

Bonitinho, n√£o?

Agora temos outros elementos para interpretar. Note que, de cara, podemos notar que muitas das rela√ß√µes s√£o claramente n√£o lineares, que existem alguns valores discrepantes para algumas vari√°veis e que a distribui√ß√£o de frequ√™ncias de boa parte das vari√°veis √© altamente assim√©trica. Neste contexto, eu diria que a correla√ß√£o de Pearson n√£o deve ser uma boa escolha, de forma que podemos refazer o painel.

Uma possibilidade seria transformar os dados (por log, por exemplo); a outra seria usar a correla√ß√£o n√£o param√©trica de Spearman.

Voc√™ pode transformar os dados por log usando a fun√ß√£o log ou log10. E se quiser calcular a correla√ß√£o de Spearman, basta adicionar o argumento method="spearman" na fun√ß√£ocor(voc√™ pode fazer isso usando a fun√ß√£o livremente ou alterando a sua chamada dentro depanel.cor`).
